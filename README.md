# 3x3 Customizable hotbar

This mod adds a 3x3 customizable hotbar to the game. This is for people using an mmo mouse in minecraft as it makes it far easier to see which button corresponds to which slot on the hotbar.

## Dependencies
This mod requires the following mods to be installed
- Fabric (Modloader)
- [Cloth Config API](https://modrinth.com/mod/cloth-config) (For configurable options)
- [ModMenu](https://modrinth.com/mod/modmenu) (So you can actually change those options)

## Options:
Hotbar Mode:
- Vanilla: The how the hotbar usually looks in minecraft
- 3x3: 3x3 hotbar, but the slots up to down, then left to right
- 3x3 (sequential): 3x3, with the slots going left to right before up to down.

Hotbar Position:
There are hotbar positions like bottom left that will place the hotbar on any of the corners of your screen. In addition, there are the following positions:
- Bottom Middle: Experience Bar is positioned further down while the hotbar will stick above it
- Bottom Middle (Compact): Health and Hunger bar are moved to the side to allow to come a bit further down.

Move UI elements to the bottom:
Only used for the corner positions. Moved the remaining elements above the hotbar down to the edge of the screen

Horizontal and Vertical Offset:
Only used while hotbar position is on one of the corners. Give the hotbar a few pixel offset from the corner of the screen.
